Latest Project

My latest project is an AI assistant which was named KIONIFY designed to help diagnose Murex-related error screenshots and analyze trade tickets. It allows users to upload error images or trade tickets, then uses AI to identify the error type, root cause, and provide step-by-step troubleshooting. The system extracts key trade ticket information like pricing fields, instrument type, and trade dates, and generates an analysis report. It also stores error categories, explanations, and ticket history in a SQL database, assisting users with Murex functional guidance and relevant UNIX commands for resolution.
